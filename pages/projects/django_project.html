Django REST Blog API

Bootcamp project: Full CRUD blog API with PostgreSQL, authentication, pagination.

Python 3.11 Django 5.0 DRF PostgreSQL JWT Auth

Features

  • ✅ User registration/login with JWT
  • ✅ Post CRUD operations
  • ✅ Comment system
  • ✅ Pagination & filtering
  • ✅ Rate limiting
  • ✅ PostgreSQL with Django ORM

Sample API Endpoints

POST /api/auth/register/
POST /api/auth/login/
GET  /api/posts/?page=1
POST /api/posts/
GET  /api/posts/{id}/
